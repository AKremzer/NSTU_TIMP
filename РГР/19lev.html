19. Grid Areas

1/20

Уровень

Каждому элементу, содержащемуся в сетке, может быть присвоено название области с помощью grid-area. Размещение именованных зон на сетке происходит в элементе контейнера с помощью свойства grid-template-areas.

Когда в элементе определяется grid-template-areas, создается сетка из ячеек, где каждая ячейка представляет собой область. Затем для каждой ячейки может быть назначено название области, для чего нужно указать расположение области в сетке. Это делается с помощью значений, представляющих имена областей, разделенных пробелами или символами, представляющими пустое пространство. Каждое значение соответствует одной ячейке в сетке. grid-template-areas позволяет гибко управлять содержимым сетки.

Вам пришел макет сайта с обозначением именованных зон. Используйте grid-template-areas, чтобы в точности отобразить его в рабочей области.
