4. Текст. Выравнивание

1/20

Уровень

Для выравнивания текста по горизонтали используется свойство text-align. Значениями этого свойства иогут быть:
left: выравнивание по левому краю.
right: выравнивание по правому краю.
center: выравнивание по центру.
justify: выравнивание текста по ширине элемента.

Свойство text-align-last позволяет определить отдельное выравнивание для последней строки текста. Значения этого свойства совпадают со значениями text-align.

Для выравнивания текста или элементов по вертикали относительно соседних элементов используется свойство vertical-align. Некоторые значения этого свойства:
top: выравнивание по верхней границе элемента.
middle: выравнивание по середине элемента.
bottom: выравнивание по нижней границе элемента.
baseline: выравнивание по базовой линии элемента.

Некоторые языки требуют написания справа налево. За это отвечает свойство direction, значение по умолчанию для которого - ltr (слева направо), rtl же выставляет направление текста справа налево. В комбинации с unicode-bidi: bidi-override; это свойство позволяет добиться "перевернутого текста".
