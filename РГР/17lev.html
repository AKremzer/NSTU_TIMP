17. Grid. Введение

1/20

Уровень

В отличие от Flexbox, который работает в одной измерении (либо горизонтально, либо вертикально), Grid Layout позволяет контролировать размещение элементов и по горизонтали, и по вертикали одновременно. Отображение содержимого контейнера в виде сетки задает display: grid. Сетка Grid состоит из следующих элементов:

1) Строки (Rows) - горизонтальные линии элементов в сетке.
2) Столбцы (Columns) - вертикальные линии элементов в сетке.
3) Линии (Grid Lines) - линии сетки между строками (row lines) или столбцами (column lines). Отсчет линий начинается с границ сетки.
4) Отступы (Gaps) - расстояние между каждым столбцом/строкой.

Одним из преимуществ Grid является возможность создания сложных макетов с использованием повторяющихся паттернов, выделением отдельных элементов сетки и так далее. Стоит понимать, что Grid - не просто жесткая двумерная конструкция, а мощный инструмент позволяющий контролировать размещение множества объектов на одной странице.

Первые свойства, с которых начнется погружение в Grid - grid-column-start, grid-column-end, grid-row-start и grid-row-end. Разберем каждое по порядку:

1) grid-column-start задает начало элемента на определенной линии между столбцами.
2) grid-column-end задает конец элемента на определенной линии между столбцами. Например, в сетке 3х3 будет 4 линии столбцов, и при задании grid-row-start: 1, grid-row-end: 3; элемент займет 2 ячейки сетки в первой строке.
3) grid-row-start задает начало элемента на определенной линии между строками.
4) grid-row-end задает конец элемента на определенной линии между строками.
