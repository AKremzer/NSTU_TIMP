2. Цвет. Продолжение

1/20

Уровень

Названия цветов не предоставляют полной творческой свободы, поэтому в этом пункте будут рассмотрены способы задания цвета:

1)RGB
Цвет может быть задан в виде rgb(значение красного, значение зеленого, значение синего), где каждое значение отвечает за интенсивность соответствующего цвета. Например, красный цвет также можно задать как rgb(255,0,0) или rgba(255,0,0,1), где последнее значение отвечает за прозрачность цвета (1 - непрозрачный, 0 - абсолютно прозрачный).
Новый вариант синтаксиса rgb в CSS объединяет rgb и rgba в одну функцию, вводя необязательное значение прозрачности и убирая запятые между значениями: rgb(255 0 0) для красного цвета или rgb(255 0 0 / .8) для красного с прозрачностью 80%;

2) HEX
Шестнадцатеричный цвет задается в формате #ККЗЗСС, где КК (красный), ЗЗ (зеленый) и СС (синий) являются шестнадцатеричными числами, отвечающими за интенсивность цвета. Красный цвет можно задать как #FF0000, где FF = 255. Еще один вариант - #ККЗЗССАА, где АА - шестнадцатеричное значение прозрасности цвета.

3) HSL
Еще один способ задания цвета в CSS - использование функции hsl() или hsla() (с прозрачностью).
HSL(Hue, Saturation, Lightness) определяет расположение цвета на цветовом круге или оттенок (Hue), его насыщенность (Saturation) и яркость (Lightness). HSLA добавляет к оттенку, насыщенности и яркости прозрачность цвета, которая определяется в процентах.

Свойство background-color задает цвет фону элемента. Используя это свойство и пройденный материал, задайте цвет фона текста как цвет с оттенком 36, 100%-ным значением насыщенности и яркостью в 30%, а цвет самого текста - с интенсивностью зеленого равной 180, а также максимальной интенсивностью для красного и минимальной интенсивностью для синего цвета.
