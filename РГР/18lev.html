18. Grid. Формат строк и столбцов.

1/20

Уровень

Определять структуру сетки более расширенно можно с помощью следующих свойств:

1) grid-template-rows позволяет определить размеры и распределение строк в сетке. Размеры каждой строки можно задать явно, используя конкретные значения (например, 100px, 50%), или использовать ключевые слова (например, auto, min-content, max-content) для динамического определения размеров строк в зависимости от содержимого.
2) grid-template-columns работает аналогично предыдущему свойству, но уже для строк.

Количество перечисленных значений этих свойств задаст количество строк/столбцов. Для обозначения размера столбцов/строк также часто используется единица fr - фракционная единица, равная 1 части доступного пространства в сетке.

Свойства выравнивания содержимого контейнера justify-content и align-content также работают для сеток. Принципы работы данных свойств с различными значениями аналогичны пройденным ранее в пункте 16.

Используя только grid-template-rows и grid-template-columns, сделайте сетку, соответствующую следующим требованиям:
1) Количество столбцов - 4, все одинакового размера (используйте фракционную единицу).
2) Количество строк - 2, первая должна занимать 30% сетки, а вторая - 60%.
3) Оставшиеся 10% места в сетке должны быть распределены равномерно по вертикали (используйте одно из свойств с прошлых уровней).
