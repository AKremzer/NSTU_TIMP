20. Бонус: Developer Tools

1/20

Уровень

Инструменты разработчика (Developer Tools) предоставляют мощный набор инструментов для разработки и отладки веб-страниц. Доступ к Developer Tools можно получить из меню при нажатии правой кнопкой мыши в любом месте веб-страницы, в пункте Исследовать элемент/Просмотреть код или похожем.

С помощью интсрументов разработчика вы можете исследовать и модифицировать CSS-свойства, просматривать стили элементов, отображать изменения в реальном времени и многое другое. Это существенно упрощает процесс разработки и отладки CSS, позволяя вам экспериментировать с различными стилями, оптимизировать внешний вид вашей веб-страницы и решать возникающие проблемы с оформлением.

Для того, чтобы просмотреть стили, применяемые к элементу на странице, необходимо открыть инструменты разработчика на интересующей странице, нажать на нужный элемент в HTML-коде страницы, после чего в нижней части инструментов разработчика появится вкладка со всеми стилями, действующими на объект.

Данный урок - завершающий, поэтому задание будет простым. С помощью Developer Tools исследуйте HTML этой страницы и найдите элемент с id "findme". Два свойства этого элемента мешают ему появиться в рабочей области - одно из этих свойств должно быть приведено к значению по умолчанию для элемента данного типа, а подсказка к значению второго содержится рядом с местонахождением findme.
