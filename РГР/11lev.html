11. Границы

1/20

Уровень

Границы border - последний компонент блочной модели, который будет рассмотрен. border задает "рамку" вокруг элемента, каждая из сторон которой может быть настроена отдельно с помощью свойств border-left, border-right, border-top и border-bottom. Каждое из этих свойств является сокращенным способом задания 3 основных свойств границы:

1) border-width задает толщину границы. Толщина может быть задана в любых единицах из пункта 7.
2) border-style задает стиль границы. Доступные стили приведены ниже

3) border-color задает цвет границы. Цвет может быть задан любым способом из приведенных в пунктах 1-2.

На границу элемента влияет padding, но не margin, так как она находится между двумя этими компонентами.
Все свойства, входящие в border, имеют значение по умолчанию, поэтому необязательно указывать значения всех свойств. Однако, для того, чтобы граница стала видимой, обязательным свойством для задания является border-style (значением по умолчанию для него является none).

Картина, которую вы вешали на стену, лишилась рамки. Подберите стиль границы, больше всего напоминающий выступающую раму, толщиной в 20 пикселей и цветом sienna.
