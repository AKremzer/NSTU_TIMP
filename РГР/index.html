1. Цвет

Цвет в CSS может задаваться несколькими способами:

1) Названия цветов
Самые распространенные цвета могут быть выбраны с помощью их названий: red, green, blue. Полный список доступных цветов представлен здесь

2) RGB
Цвет может быть задан в виде rgb(красный, зеленый, синий), где каждое значение отвечает за интенсивность соответствующего цвета. Например, красный цвет также можно задать как rgb(255,0,0)

3) HEX
Шестнадцатеричный цвет задается в формате #ККЗЗСС, где КК (красный), ЗЗ (зеленый) и СС (синий) являются шестнадцатеричными числами, отвечающими за интенсивность цвета. Красный цвет можно задать как #FF0000, где FF = 255

Цвет фона задается с помощью background-color: цвет;
Для цвета текста используется color: цвет;

Текст в правой части экрана пока выглядит довольно серо. Допишите код так, чтобы добавить цвета фону и тексту на нем. Не бойтесь экспериментировать!
