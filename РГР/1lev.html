1. Введение в CSS. Цвет

1/20

Уровень

CSS, Cascading Style Sheets - язык стилей, который используется для описания внешнего вида веб-страниц. Это одна из основных технологий, применяемых в современной веб-разработке, позволяющая разделить структуру страницы, за которую отвечает HTML, от ее внешнего представления и стилизации.

Синтаксис CSS включает в себя селектор, указывающий на HTML-элемент, к которому применяется стиль, блок объявлений, состоящий из пар свойство-значение, где свойствам CSS задаются подоходящие значения, определяющие уникальный внешний вид элемента.

Первое свойство, с которого начнется погружение в CSS - color.
color определяет цвет текста, который содержит элемент.

Цвет в CSS может задаваться несколькими способами, которые будут подробно рассмотрены в следующем пункте. Самым простым из них является задание цвета с помощью его названия - CSS поддерживает 147 цветов, от распространенных (yellow, red) до специфичных (lawngreen, orchid и так далее). Здесь представлено разнообразие цветов, которые могут быть заданы ключевыми словами.
